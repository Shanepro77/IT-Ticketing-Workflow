# IT-Ticketing-Workflow

## Overview
A real-world help desk simulation using a free ticketing system (Spiceworks / Jira Free). Demonstrates how I diagnose, document, and resolve user issues.

## Skills Demonstrated
- Ticket creation and classification
- Troubleshooting methodology
- SLA-based prioritization
- Documentation of resolutions
- User communication skills

## Included
- Sample tickets with full troubleshooting steps
- Screenshots of ticket system
- My help desk documentation style
